Marcos de trabaj de Java

La palabra inglesa "DRAMEWORK" (marco de trabajo) define, en terminos generales, un conjunto estandarizado de conceptos, practicas y criterios para enfocar un timpo de problematica particular que sirve como referencia, para enfrentar y resolver nuevos problemas de idole similar

En el desarrollo de software, un framework o infrestructura digital, es una estructura conceptual y tecnologiaca de soporte definido, normalmente con artefactos o modulos de software concretos, que puede servir de base la organizacion y desarrollo de software, Tipicamente hablando, puede incluir soporte de programas, bibliotecas, y un lenguaje interpretado, entre otras herramientas, para asi ayudar a desarrollar y unir componentes de un proyecto.
